## This app searches for corporate offices close to a typed localization  

To view the app in AI Studio: https://ai.studio/apps/drive/1wUJrlC1zYhCZg84f6h9xBxUCiiPh4hKk

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
